# Nombre del Proyecto: Proyecto3_Port

Descripción breve del proyecto: Este es un proyecto de ejemplo usando Vite que muestra una aplicación React con múltiples componentes y estilos. Es útil para demostrar la carga perezosa de componentes y el manejo de estados dentro de una aplicación SPA (Single Page Application).

## Instalación

Para instalar las dependencias del proyecto, ejecuta el siguiente comando:

```bash
npm install
```

## Uso

Para correr el servidor de desarrollo y ver la aplicación en acción, utiliza:

```bash
npm run dev
```

## Estructura del Proyecto

El proyecto contiene varios componentes organizados de la siguiente manera:

- `App.jsx`: Componente principal que maneja la lógica de visualización de otros componentes.
- `Navbar`, `Header`, `Sidebar`: Componentes para la navegación y estructura básica de la página.
- `SecondPage.jsx`, `ThirdPage.jsx`, `FourthPage.jsx`: Componentes que representan diferentes páginas de la aplicación.
- `App.css`, `index.css`, `Sidebar.css`, `SecondPage.css`, `ThirdPage.css`, `FourthPage.css`: Archivos de estilo para la aplicación.
- Componentes adicionales como `Card`, `Carousel` y otros, gestionan la presentación de contenido específico.

## Scripts Disponibles

En este proyecto, puedes ejecutar:

- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Compila la aplicación para producción.
- `npm run lint`: Ejecuta ESLint para asegurar la calidad del código.
- `npm run preview`: Sirve la versión de producción de la aplicación localmente.

## Dependencias

Este proyecto utiliza varias bibliotecas de terceros como:

- `react`
- `react-dom`
- `better-react-carousel`
- `react-scroll`
- `react-slick`
- `slick-carousel`
- `swiper`
